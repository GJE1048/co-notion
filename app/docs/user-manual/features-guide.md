## 文档编辑

### 富文本编辑器

平台提供了强大的富文本编辑器，支持多种格式和元素。

#### 基本格式

**文本格式**
- **粗体**: `Ctrl+B` 或点击工具栏按钮
- **斜体**: `Ctrl+I` 或点击工具栏按钮
- **下划线**: `Ctrl+U` 或点击工具栏按钮
- **删除线**: `Ctrl+Shift+X`
- **代码**: `Ctrl+E` 选中文字后按，或 `` ` `` 包裹

**标题级别**
- 一级标题: `Ctrl+Alt+1`
- 二级标题: `Ctrl+Alt+2`
- 三级标题: `Ctrl+Alt+3`
- 依次类推到六级标题

**列表**
- 无序列表: `Ctrl+Shift+8` 或 `-` + 空格
- 有序列表: `Ctrl+Shift+7` 或 `1.` + 空格
- 任务列表: `- [ ]` 或 `- [x]`

#### 插入元素

**链接**
```
方法1: 选中文字后按 Ctrl+K
方法2: 点击工具栏链接按钮
方法3: 输入 [链接文字](URL)
```

**图片**
```
拖拽图片文件到编辑器
或点击工具栏图片按钮选择文件
或粘贴图片 (Ctrl+V)
```

**表格**
```
方法1: 点击工具栏表格按钮
方法2: 使用 Markdown 语法

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据 | 数据 | 数据 |
```

**代码块**
```javascript
// 点击工具栏代码块按钮
// 或使用 ``` 包裹

function hello() {
  console.log('Hello, World!');
}
```

**数学公式**
```
行内公式: $E = mc^2$
块级公式: $$E = mc^2$$
```

**引用**
```
> 这是一段引用文字
```

**分割线**
```
---
或
***
```

### Markdown 支持

平台完全支持 Markdown 语法，可以在编辑器中混合使用可视化编辑和 Markdown。

**常用 Markdown 语法**:

```markdown

## 二级标题

**粗体** *斜体* `代码`

- 无序列表项
- 另一个列表项

1. 有序列表项
2. 另一个列表项

[链接文字](https://example.com)

![图片描述](image-url)

> 引用文字

`行内代码`

\`\`\`javascript
代码块
\`\`\`
```

## 协作功能

### 实时协作编辑

#### 开始协作

1. **邀请协作者**
   - 点击文档右上角的"分享"按钮
   - 输入协作者邮箱地址
   - 选择权限级别

2. **查看协作者状态**
   - 在线协作者显示绿色头像
   - 离线协作者显示灰色头像
   - 查看协作者当前编辑位置

3. **实时同步**
   - 所有协作者的编辑实时同步
   - 看到其他人的光标位置和选择范围
   - 支持同时多人编辑

#### 协作最佳实践

**沟通协调**
- 使用评论功能进行讨论
- 在编辑前通知其他协作者
- 定期检查协作者状态

**冲突避免**
- 不同协作者编辑不同段落
- 使用"正在编辑"指示器
- 及时保存和同步

### 评论系统

#### 添加评论

**方法1: 选择文本评论**
```
1. 选中需要评论的文本
2. 右键选择"添加评论"
3. 输入评论内容
4. 点击"发布"
```

**方法2: 通用评论**
```
1. 点击工具栏评论按钮
2. 选择评论位置
3. 输入评论内容
```

#### 评论管理

**回复评论**
- 点击评论下的"回复"按钮
- 输入回复内容

**解决评论**
- 点击评论右侧的"✓"按钮
- 已解决的评论会折叠显示

**编辑/删除评论**
- 点击评论菜单 (⋯)
- 选择编辑或删除

**评论过滤**
- 查看所有评论
- 只看未解决的评论
- 只看我的评论

### 通知系统

#### 通知类型

- **文档更新**: 协作者修改了文档
- **新评论**: 有人回复了你的评论
- **权限变更**: 文档权限发生变化
- **协作者加入**: 新协作者加入文档

#### 通知管理

**查看通知**
- 点击顶部通知铃铛图标
- 查看所有未读通知

**通知设置**
- 设置 → 通知偏好
- 选择通知类型和频率
- 配置邮件通知

## AI智能辅助

### AI续写

#### 使用场景

- 文章续写
- 思路扩展
- 内容填充

#### 操作步骤

1. **选择文本**
   - 选中要续写的段落末尾

2. **触发续写**
   - 右键菜单: "AI续写"
   - 快捷键: `Ctrl+Shift+C`
   - 工具栏: AI按钮 → 续写

3. **设置参数**
   ```
   风格: 正式/轻松/学术/创意
   长度: 简短(100字)/中等(300字)/详细(500字)
   语气: 客观/主观/专业
   ```

4. **预览和应用**
   - 查看AI生成的内容
   - 编辑修改后应用
   - 或重新生成

#### 续写技巧

**指定上下文**
- 选择更多上下文文本
- AI会基于更完整的语境续写

**使用提示词**
- 在选中文本前添加提示
- 例如: "[续写: 详细说明技术实现]"

### AI润色

#### 润色类型

**语法优化**
- 修复语法错误
- 改善句子结构
- 统一标点符号

**表达优化**
- 提升语言表达
- 增强可读性
- 调整语境适应性

**风格转换**
- 正式 ↔ 轻松
- 学术 ↔ 通俗
- 中文 ↔ 英文

#### 润色操作

1. **选择文本**
   - 选中需要润色的段落

2. **选择润色类型**
   - 工具栏: AI按钮 → 润色
   - 选择具体类型

3. **应用润色**
   - 对比原文本和润色结果
   - 接受或拒绝修改
   - 局部应用

#### 摘要类型

**提取式摘要**
- 从原文提取关键句子
- 保持原文措辞
- 准确度高

**生成式摘要**
- 重新组织生成摘要
- 语言更加流畅

#### 摘要参数

```

重点关注: 整体/开头/结尾/特定段落
```

## 搜索功能

### 基本搜索

#### 搜索范围

- **全部内容**: 文档标题和内容
- **仅标题**: 只搜索文档标题
- **标签**: 按标签筛选
- **作者**: 按创建者或修改者

#### 搜索语法

**精确匹配**
```
"精确的短语"
```

**排除关键词**
```
关键词 -排除词
```

**标签搜索**
```
tag:标签名
```

**作者搜索**
```
author:用户名
```

**时间范围**
```
modified:2024-01-01..2024-12-31
created:>2024-01-01
```

**组合搜索**
```
项目管理 author:张三 tag:重要
```

## 权限管理

### 权限级别

#### 所有者权限

- 完全控制文档
- 管理所有协作者权限
- 删除文档
- 转让所有权

#### 管理员权限

- 编辑文档内容
- 管理协作者 (除所有者转让)
- 修改文档设置
- 删除文档

#### 编辑者权限

- 编辑文档内容
- 添加/回复评论
- 查看协作者列表
- 无法管理权限

#### 评论者权限

- 查看文档
- 添加/回复评论
- 无法编辑内容

#### 查看者权限

- 只读查看文档
- 无法编辑或评论

### 权限设置

#### 邀请协作者

1. **打开分享面板**
   - 点击"分享"按钮

2. **添加用户**
   - 输入邮箱地址
   - 选择权限级别

3. **发送邀请**
   - 用户会收到邮件
   - 点击链接接受邀请

#### 批量邀请

- 上传包含邮箱列表的CSV文件
- 批量设置权限
- 发送群组邀请

#### 权限变更

**修改权限**
- 在协作者列表中选择用户
- 更改权限级别
- 确认变更

**权限历史**
- 查看权限变更记录
- 了解变更原因和时间

### 安全控制

#### 二次确认

**高风险操作**
- 删除文档
- 更改权限
- 永久删除

**确认方式**
- 弹窗确认
- 邮箱验证码
- 应用内确认
