# 快速开始指南

欢迎使用智能化多人文档编辑平台！本指南将帮助你快速上手，了解平台的基本功能和使用方法。

## 账号注册与登录

### 注册新账号

1. **访问注册页面**
   - 打开浏览器，访问平台网址
   - 点击页面右上角的"注册"按钮

2. **填写注册信息**
   ```
   邮箱地址: your-email@example.com
   用户名: your-username (可选)
   显示名称: Your Display Name
   密码: 至少8位，包含字母和数字
   ```

3. **验证邮箱**
   - 检查邮箱中的验证邮件
   - 点击验证链接完成注册

4. **完善个人资料** (可选)
   - 上传头像
   - 设置个人偏好 (主题、语言等)

### 登录账号

1. **访问登录页面**
   - 点击页面右上角的"登录"按钮

2. **选择登录方式**
   - **邮箱登录**: 输入邮箱和密码
   - **第三方登录**: 支持 Google、GitHub 等 (如果启用)

3. **安全选项**
   - 启用双因子认证 (推荐)
   - 设置登录提醒

## 创建你的第一个文档

### 1. 创建新文档

**方法一: 从首页创建**
- 登录后，在首页点击"新建文档"按钮
- 输入文档标题
- 选择文档类型 (普通文档/模板)

**方法二: 从文件夹创建**
- 在左侧文件夹列表中右键点击文件夹
- 选择"新建文档"
- 或点击文件夹内的"新建文档"按钮

### 2. 编辑文档

**基本编辑**
- 点击文档标题进入编辑模式
- 开始输入内容
- 支持 Markdown 语法

**格式设置**
- 使用工具栏进行格式化
- 快捷键支持:
  - `Ctrl+B`: 粗体
  - `Ctrl+I`: 斜体
  - `Ctrl+U`: 下划线
  - `Ctrl+K`: 链接

**插入元素**
- **图片**: 拖拽上传或点击插入按钮
- **表格**: 使用表格工具或 Markdown 语法
- **代码块**: 选择语言高亮
- **数学公式**: 支持 LaTeX 语法

### 3. 保存文档

文档会自动保存，你也可以：
- 点击保存按钮手动保存
- 查看保存状态指示器
- 查看版本历史

## 组织你的文档

### 创建文件夹

1. **在根目录创建**
   - 点击左侧边栏的"新建文件夹"按钮
   - 输入文件夹名称

2. **在现有文件夹中创建**
   - 右键点击父文件夹
   - 选择"新建子文件夹"

### 移动和整理文档

**移动文档**
- 拖拽文档到目标文件夹
- 或右键选择"移动到"

**添加标签**
- 在文档编辑页面点击"标签"按钮
- 输入标签名称
- 或从现有标签中选择

**搜索文档**
- 使用顶部搜索框
- 支持关键词搜索
- 按标签、作者、时间筛选

## 邀请协作者

### 添加协作者

1. **打开文档分享设置**
   - 在文档编辑页面点击"分享"按钮
   - 或右键文档选择"分享"

2. **邀请用户**
   - 输入协作者的邮箱地址
   - 选择权限级别:
     - **查看者**: 只能查看文档
     - **评论者**: 可以查看和添加评论
     - **编辑者**: 可以编辑文档内容
     - **管理员**: 可以管理协作者权限

3. **发送邀请**
   - 点击"发送邀请"
   - 协作者会收到邮件通知

### 管理协作者

**查看协作者列表**
- 在分享面板查看所有协作者
- 查看他们的权限级别和加入时间

**修改权限**
- 点击协作者名称旁边的权限下拉菜单
- 选择新的权限级别

**移除协作者**
- 点击协作者旁边的"移除"按钮
- 确认移除操作

## 开始协作编辑

### 实时协作

1. **邀请协作者加入**
   - 按照上述步骤添加协作者

2. **查看在线状态**
   - 在文档顶部查看协作者头像
   - 绿色圆点表示在线
   - 灰色圆点表示离线

3. **实时编辑**
   - 多名协作者可以同时编辑
   - 看到其他人的光标位置
   - 查看编辑历史

4. **解决冲突**
   - 平台自动处理大部分冲突
   - 如有冲突，会提示手动解决

### 添加评论

1. **选择文本**
   - 选中需要评论的文本

2. **添加评论**
   - 右键选择"添加评论"
   - 或点击工具栏的评论按钮

3. **回复评论**
   - 点击评论查看详情
   - 点击"回复"添加回复

4. **管理评论**
   - 解决评论
   - 编辑或删除评论

## 使用AI智能功能

### AI续写

1. **选择文本**
   - 选中要续写的段落末尾

2. **触发AI续写**
   - 右键选择"AI续写"
   - 或使用快捷键 `Ctrl+Shift+C`

3. **调整参数**
   - 选择续写风格 (正式/轻松/学术)
   - 设置续写长度 (简短/中等/详细)

4. **应用结果**
   - 预览AI生成的内容
   - 点击"应用"插入文档

### AI润色

1. **选择文本**
   - 选中需要润色的段落

2. **选择润色类型**
   - 语法优化
   - 表达优化
   - 结构优化

3. **应用润色**
   - 预览润色结果
   - 接受或拒绝修改

### 其他AI功能

**翻译文本**
- 选择语言进行翻译
- 支持中英文互译

**生成摘要**
- 为长文档生成摘要
- 选择摘要长度和类型

**智能问答**
- 基于文档内容回答问题
- 提供相关内容引用

## 常见问题

### 无法登录怎么办？

**检查网络连接**
- 确保网络连接正常

**清除浏览器缓存**
- 清除浏览器缓存和Cookie
- 尝试使用无痕模式

**重置密码**
- 点击"忘记密码"
- 按邮件指示重置

### 文档无法保存？

**检查权限**
- 确认你有编辑权限

**检查网络**
- 确保网络连接稳定

**查看错误信息**
- 查看页面底部的错误提示
- 刷新页面重试

### 协作者无法看到我的编辑？

**检查协作者状态**
- 确认协作者已接受邀请
- 查看协作者在线状态

**刷新页面**
- 让协作者刷新浏览器页面

**检查权限设置**
- 确认协作者有适当的权限
